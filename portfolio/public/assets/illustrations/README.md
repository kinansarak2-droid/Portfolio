# Illustrations

Drop custom illustrator exports here.

## Subfolders
- `icons/` — Small icon-scale illustrations (24px–64px). Export as SVG.
- `scenes/` — Large scene illustrations used in sections. Export as SVG or WebP.

## Export Tips (Adobe Illustrator)
- Use "Export As" → SVG for vector illustrations
- Enable "Responsive" in SVG options to remove fixed width/height
- Optimize with SVGO before dropping in

## Naming Convention
`icon-[name].svg`, `scene-[section].svg`
